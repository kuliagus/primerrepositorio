El nombre de Index es buscado por los servidores como nombre de entrada. 
Hay que respetar la apertura que es 
1    <!DOCTYPE html>  <--- es la primer apertura que se debe tener.

Las etiquetas de apertura empiezan por ejemplo:
1   <html>    
y cierran por ejemplo:
1   <html>   </html>  ó  <p>  </p> etc.
La barra determina el cierre.

El elemento <head> hace que podamos agregarle en su encabezado información sobre la página y metadatos.
Es decir, va a hacer que sea más facil buscarla por los motores de busqueda. (search)

La etiqueta <body> es en donde vamos a tener el cuerpo de la página. 
Ejemplo: HTML (estructura) , CSS (presentación/apariencia) , JavaScript (Dinamismo/interacción)

El elemento <title> es lo que visualizamos en la pestaña.

El elemento <h1> da el título principal a la página, no a la pestaña. 'H' de Headed.
Puede haber más subgéneros de elementos h1. Ej: <h2> ; <h3> , etc. hasta el <h6>.
Estos elementos <h1> hasta el <h6> son los que diagraman el contenido en distintos tamaños.

El elemento <p> de paragraph (párrafo) crea uno.

°Los elementos deben estar legibles para una fácil lectura, sea por uno mismo, o por un compañero/equipo.

VCODE atajo: CTRL + SHFT + P = '>format Document' (enter) y corrige el código.

Hacemos click derecho en el index.html y podemos agregar Live Preview para visualizar al instante las modificaciones del código.

Para escribir un enlace tenemos que escribir la etiqueta <a href="acá va el valor del atributo"></a> 
la 'a' es de Atributo. Podemos pegar un enlace https para que nos lleve ahí. Y antes de cerrar le ponemos el Título. Ejemplo: Ir a Google
Esto nos manda a otra página pero desde la misma pestaña. (impráctico)
¿Cómo hacer que abra una nueva pestaña este enlace? 
Agregando el atributo: target="_blank" (ó)  _self
Podemos titular el "Ir a Google" con un <h2>Enlaces</h2> arriba.

¿Qué son las listas? Agrupación de elementos.
Le damos nombre con <h2>
Etiquetamos: <ul>  (Unordered List) </ul> Esto crea una lista DESORDENADA
{Dentro de este bloque agregamos la etiqueta <li>} List Item.
Los ítems tienen un circulo negro. Podemos agregar los que querramos.

Para hacer una lista ORDENADA utilizamos <ol></ol>
Los ítems <li> van a tener números. Todo elemento que contenga <li> va a tener un orden.

En el caso de las imágenes, es un elemento que se cierra solo. Autoconclusivo.
Se abre con '<img src=' y se cierra con '/>' 
'src=""' es de dónde vamos a renderizar la imágen (puede ser local , es decir guardada , o URL )
'alt=""' el fin de este elemento es leer en voz alta lo que se inserte.
EJEMPLO: <img src="acá copiamos la URL o imágen local" alt="acá va el texto a escoger" />
Cuando se rompe una imágen (es decir le falta la imágen a la URL) aparece un ícono de foto en blanco con el texto.

EJEMPLO SIN URL: Selecciona una imágen de internet, la guarda en la carpeta que esté en VCODE
como en este caso 'index.html', le pone un <h3> declarando imágen desde una URL 
y abajo otro <h3> declarando la ímagen local
(acá va un <img src=" .... "  alt=" ... ">

Creamos una Carpeta, la nombramos y vamos a mover la imágen a la carpeta, y ya no está más a nivel HTML sino, dentro de la carpeta.
Se rompe la imágen y no la muestra.

¿Cómo usar un comentario? Es una porción de código que no se muestra.
EJEMPLO: <!---Este elemento renderiza una imagen desde una URL--->
Los comentarios sirven para que quien toque el código, sepa el porqué.

Podemos solucionar la imágen que estaba rota colocando Archivos.
EJEMPLO: <img src="archivos/messi2.jpg" alt="Messi 2">

Para navegar hacia otra pagina podemos crear un nuevo .html (lo creamos ingresando html:5) 
Le borramos la metadata (son 2 lineas códigos: la de http-equiv y name="viewport") y la titulamos como queremos
Para volver hacia atrás podemos agregar debajo de <body> un elemento <a href="index.html">Ir al index</a>

Utilizar el elemento de una etiqueta <br/> entre los '<a href=' hace que los muestre uno debajo del otro.

ELEMENTOS ANIDADOS: elementos dentro de otros elementos.
creamos un elemento <a href="pegamos el link a donde queremos que nos lleve"></a>
Podemos reemplazar el !comentario borrandoló e insertando elemento img
Qudaría: img src="archivos/messi2.jpg" alt="Link a wikipedia de Messi"
(dentro del elemento <a></a>)

PD: un <ul> puede ir dentro de un <ol> y biceversa.
